## Create cart

`/cart/create-cart`

#### Parameters

None

#### Example

`pp://staging/cart/create-cart`

---


## Add item to cart

`/cart/add-item`

#### Parameters

|Name|Required|Type|
|---|---|---|
|cartId|Yes|string|
|skuId|Yes|number|
|quantity|Yes|number|
|offerId|No|number|

#### Example


`pp://staging/cart/add-item?cartId=1349&skuId=502&quantity=2&offerId=1928`

---


## Remove item from cart

`/cart/remove-item`

#### Parameters

|Name|Required|Type|
|---|---|---|
|cartId|Yes|string|
|skuId|Yes|number|
|quantity|Yes|number|
|offerId|No|number|

#### Example

`pp://staging/cart/remove-item?cartId=1349&skuId=184184&quantity=1&offerId=8`

---

## Set customer

`/cart/set-customer`

#### Parameters

|Name|Required|Type|
|---|---|---|
|cartId|Yes|string|
|no|Yes|string|

Or

|Name|Required|Type|
|---|---|---|
|cartId|Yes|string|
|brandCode|Yes|string|
|mobile|Yes|string|

#### Example

`pp://staging/cart/set-customer?cartId=1349&no=EE0000053147`

Or

`pp://staging/cart/set-customer?cartId=1349&brandCode=EE&mobile=18512345678`
